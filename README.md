# Getting Started
  - Clone: `git clone https://github.com/kayy0812/simpleNodeTools.git`.
  
## Setup
  - `npm install`.
  - Put your accessToken in `simpleMain.js`.
  
## Using
  - `node simpleLikes` → Like all posts from users (Delay time: 2s)
  - `node simpleUser` → Get all user information
  - _updating..._
